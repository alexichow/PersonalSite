@Save_On_Games Automated Twitter Account
Python, PRAW, Twython
Source Code • @Save_On_Games

A Python script that uses PRAW to collect the most recent 25 post from r/gamedeals. The program then checks these new links against a text file containing links that have already been posted. If the links haven't been posted, the link is shortened using Google's URL shortener and posted to the Twitter account using Twython.

The program also supports sending deals directly to an email account which can be useful if looking for a specific game to go on sale.


Smith-Waterman Algorithm (Genetic Sequence Alignment)
Source Code

The Smith-Waterman algorithm is an algorithm that performs local sequence alignment for two DNA sequences. The input to the algorithm are the two sequences to be aligned, the scoring matrix to use in creating the alignment, and the gap penalty to score mismatches.

The program performs the local alignment on both genetic sequences and outputs the best-scoring alignment.
